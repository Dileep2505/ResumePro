About ResumePro
ResumePro is an AI-assisted resume platform that helps you analyze resumes, improve ATS scores, and find job matches based on your skills.

Features:
• Resume analysis and structured parsing
• Job matching with skill-gap insights
• Career exploration and market trend views
• JD-based resume optimization suggestions
• Google OAuth support for secure login

Tech Stack: FastAPI, SQLAlchemy, HTML/CSS/JS, SQLite/CSV
Deployment: GitHub Pages (frontend), Render (backend)
Source & Docs: See project README for details.
